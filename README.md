# The-Truth-Daily
The Truth Daily is a modern, responsive static news website built with HTML, CSS, and JavaScript. It features a clean green-branded header, clickable news articles, individual story pages, social media integration, and an advertisement section.

The Truth Daily is a modern news blog website focused on political, economic, and social news.  
The platform is designed to be clean, mobile-friendly, and easy to manage.

## Features
- Modern homepage layout with news previews
- Dedicated stories page for all published articles
- Individual story pages for full article content
- Responsive design (mobile, tablet, desktop)
- Social media integration
- Advertisement section for sponsors
- Custom brand header and typography

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Google Fonts
- Boxicons

## Hosting
This website is hosted for free on **Netlify** and connected directly to GitHub for continuous deployment.

## Content Management (Upcoming)
The project is structured to be integrated with **Netlify CMS**, allowing the site owner to:
- Log in securely
- Create and edit stories without coding
- Publish articles directly to the website
